Create a Project

(code for this section)

  • Make a folder for your project and cd into it.

    $ mkdir project
$ cd project

  • Create a file called Justfile and put the following into it. 

    # Shows available recipies
help:
    just --list

  • Test that using just help

    $ just help
just --list
Available recipes:
    help # Shows available recipes

  • Make file named deps.edn with the following contents.

    This is what sets the version of Clojure to use and tells Clojure that your code will be in the src folder. 

    {:paths ["src"]
 :deps {org.clojure/clojure {:mvn/version "1.12.0"}}}

  • Pick a top level namespace for your project.

    For the rest of this we will use example, but feel free to pick something else. Just replace example in all the following instructions with whatever you picked

  • Create a folder named src

    $ mkdir -p src

  • Make a src/example/main.clj file with following contents. 

    (ns example.main)

(defn -main []
  (println "Hello, world"))

  • Run that file

    $ clojure -M -m example.main
Hello, world

    The first -M means you are want to run the function named -main. The second, lower-cased, -m means you want to run the -main function in the example.main namespace.

  • Add the command to run that file to your Justfile

    # Shows available recipes
help:
    just --list
                
run:
    clojure -M -m example.main

  • Run your project with just run

    This will be very useful once running your project takes more than an easy to remember command. 

    $ just run
clojure -M -m example.main
Hello, world
After all that, you should have a project structure that looks like this. You can check using the tree command. 
src/example/
  main.clj
deps.edn
Justfile
