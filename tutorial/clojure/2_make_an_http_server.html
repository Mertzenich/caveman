Make an HTTP Server

  • Add a dependency to the ring library to your deps.edn 

    {:paths ["src"]
 :deps {org.clojure/clojure {:mvn/version "1.11.3"}
        ring/ring {:mvn/version "1.12.2"}}}

    Ring is the library most everyone in the Clojure world uses for this.

  • Edit your src/main.clj file so it has the following contents. 

    (ns main
  (:require [ring.adapter.jetty :as jetty]))

(defn handler
  [request]
  {:status 200
   :headers {"Content-Type" "text/html"}
   :body "Hello, world"})

(defn -main []
  (jetty/run-jetty #'handler {:port 9999}))

  • Run the server with just run

    You should see a warning like the following. Ignore it for now.

    SLF4J: No SLF4J providers were found.
SLF4J: Defaulting to no-operation (NOP) logger implementation
SLF4J: See https://www.slf4j.org/codes.html#noProviders for further details.

  • Open your web browser and go to http://localhost:9999
